Set up the environment before we start: we’ll use the StatsBase and OrderedCollections packages.

using StatsBase, OrderedCollections

Example: plotting vocabulary frequency in the Declaration of Independence

The following cell defines a variable named declaration with the text of the Declaration of Independence.

Let’s see how many characters it has:

Compute the frequencies

wordcounts = split(declaration) |> countmap |> OrderedDict

words = wordcounts |> keys |> collect
counts = wordcounts |> values |> collect
